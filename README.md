# Neon Snake

A cyberpunk neon snake game built with HTML, CSS, and JavaScript.

## Features

- Glowing green snake with flickering yellow food
- Speed increases every 3 points
- Player name entry and persistent top scores
- Background music and game sound effects
- Keyboard controls for laptop users
- Touch controls for mobile users
- Responsive layout for laptop and mobile screens

## Run

Open `index.html` in your browser.

## Controls

- Move: `W`, `A`, `S`, `D` or arrow keys
- Pause: `P`
- Restart: `Space`
- Mobile: on-screen touch controls
